# 一绳一愿小程序 · 产品需求文档

> 版本：v2.0
> 日期：2026-04-27
> 目标：唐卡手绳/项链展示小程序（非电商，纯展示+引流）

---

## 一、产品线结构

```
一绳一愿
├── 手绳款
│   ├── 按色系：朱砂 / 青金 / 松石 / 月白 / 墨银 / 紫檀 ...
│   └── 按老师：扎基拉姆 / 八大守护 / 药师佛 / 绿度母 / 狐仙 ...
├── 项链款
│   ├── 按色系：（同上）
│   └── 按老师：（同上）
└── 通用属性
    ├── 适合生肖
    ├── 祝福寓意（标签）
    └── 工艺说明
```

---

## 二、页面结构

### 2.1 首页
- 品牌氛围区（大图/视频背景）
- 两个入口：手绳款 | 项链款
- 不做商品瀑布流，保持高级感

### 2.2 产品列表页（手绳/项链）
- **顶部筛选栏**：
  - Tab 1：色系（全部/朱砂/青金/松石/月白/墨银）
  - Tab 2：老师/神佛（全部/扎基拉姆/八大守护/药师佛/绿度母/狐仙）
  - Tab 3：生肖（全部/鼠/牛/虎...）
- **商品卡片**：
  - 大图
  - 右下角标签：祝福寓意（如「旺财」「平安」「健康」）
  - 名称 + 价格

### 2.3 产品详情页
- 全屏产品图
- 神佛/老师介绍（适合什么生肖、什么寓意）
- 工艺说明
- 「请愿入盒」CTA（唤起客服）

### 2.4 故事页
- 各神佛传说
- 适合生肖说明
- 佩戴禁忌/注意事项

### 2.5 我的页
- 极简
- 联系客服（复制微信号 / 唤起微信客服）
- 关于我们

## 四、关键交互

| 交互 | 说明 |
|------|------|
| 筛选 | 色系/老师/生肖 三维度联动筛选 |
| 标签 | 商品图右下角显示祝福标签 |
| CTA | 「请愿入盒」→ 唤起客服 / 复制微信号 |
| 分享 | 商品页/故事页可分享 |

## 五、技术方案

### 5.1 前端
- 原生微信小程序
- 暗色主题（墨黑+鎏金）

### 5.2 后端
- 阶段一：Mock 数据（当前）
- 阶段二：微信云开发（推荐）
  - 云数据库：products, deities, stories
  - 云存储：商品图片
- 阶段三：自有后端（如需复杂管理后台）

### 5.3 维护方式
- 云开发控制台直接改数据
- 或飞书多维表 → 我帮你同步

---

## 六、待确认问题（审核红线）

1. **「旺财」等词汇**：微信审核对宗教/迷信敏感，需确认定位
2. **微信号/二维码**：小程序内不能直接放二维码引流
3. **宗教内容边界**：故事页介绍神佛传说的尺度
